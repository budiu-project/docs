## 页面 1/4：寻宝地图（主页）

### 页面信息

| 属性 | 值 |
|------|-----|
| 页面名称 | 寻宝地图 |
| 层级 | Tab 主页（底部 Tab 栏"寻宝"入口） |
| 页面用途 | 展示附近宝箱、引导用户遛狗开箱、完成开箱奖励领取 |

### 布局结构

```
┌─────────────────────────────────┐
│ [系统状态栏]                      │
├─────────────────────────────────┤
│ ┌─────────────────────────────┐ │
│ │ 顶部状态栏（半透明浮层）       │ │
│ │ [今日+85分] [3/10] [券包🔴]  │ │
│ └─────────────────────────────┘ │
│ ┌─────────────────────────────┐ │
│ │        提示条（条件触发）      │ │
│ └─────────────────────────────┘ │
│                                 │
│         全 屏 地 图              │
│                                 │
│      📦230m   💰150m            │
│              🐕 ← 用户位置       │
│   📦可开启        📦380m         │
│                                 │
│ [筛选]                   [定位] │
│                                 │
│ ┌─────────────────────────────┐ │
│ │     奖励卡片（开箱后弹出）     │ │
│ └─────────────────────────────┘ │
├─────────────────────────────────┤
│ [首页] [寻宝✓] [轨迹] [我的]    │
└─────────────────────────────────┘
```

### 元素清单

**E1: 顶部状态栏**
- 位置：安全区顶部下方，左右边距 16px
- 高度：44px
- 背景：白色 80% 不透明度，圆角 12px，底部阴影
- 子元素：
  - 左侧 — 积分进度：图标（金币 16×16）+ 文案"今日 +{n} 分"，字号 14px，色值 #333
  - 中间 — 宝箱进度：文案"{n}/10"，字号 14px，色值 #666
  - 右侧 — 券包入口：券包图标 24×24，可点击区域 44×44，有新券时右上角红点 8×8

**E2: 提示条**
- 位置：紧贴 E1 下方，左右边距 16px
- 高度：36px
- 触发条件：GPS 信号弱 / 网络异常 / 宝箱被抢走
- 样式：背景色按类型区分（警告黄 #FFF3CD / 错误红 #F8D7DA / 信息蓝 #D1ECF1），圆角 8px
- 文案字号 13px，单行显示
- 网络异常时右侧显示"重试"按钮
- 3 秒后自动消失（"宝箱被抢走"类型），或持续显示直到状态恢复（GPS / 网络类型）

**E3: 地图区域**
- 占满全屏（在状态栏和 Tab 栏之间），E1 / E2 浮于地图之上
- 地图 SDK：高德地图
- 默认缩放：以用户位置为中心，展示约 500m 半径
- 缩放范围：可放大至街道级别，缩小最远展示 1km 半径
- 用户位置标记：遛狗主题定位图标（替代默认蓝点），40×40px，带方向指示

**E4: 宝箱图标（地图 Marker）**

| 类型 | 图标尺寸 | 描述 |
|------|---------|------|
| 积分宝箱 — 常态 | 36×36 | 木质宝箱，标准显示 |
| 积分宝箱 — 可开启 | 40×40 | 变亮 + 循环弹跳动画（上下 4px，周期 0.6s） |
| 积分宝箱 — 变灰 | 36×36 | 灰度处理，opacity 50%，不可点击 |
| 商家宝箱 — 常态 | 44×44 | 金色宝箱，略大 |
| 商家宝箱 — 可开启 | 48×48 | 变亮 + 循环弹跳动画 |
| 商家宝箱 — 已领取 | 44×44 | 灰度 + 右下角绿色勾 16×16 |
| 稀有宝箱 | 对应尺寸 | 对应类型图标 + 紫色光环动画（脉冲式发光，周期 1.5s） |
| 聚合气泡 | 48×48 | 圆形，内显数字，背景色取最高优先级类型色 |

- 每个宝箱下方距离标注：字号 11px，色值 #666，背景白色 70% 不透明度圆角胶囊
- 进入 50m 后距离标注替换为"可开启"，字号 12px，色值主题绿 #34C759

**E5: 筛选按钮**
- 位置：左下角，距底部 Tab 栏上方 16px，距左边距 16px
- 尺寸：圆角胶囊按钮，高度 36px，内边距 12px
- 样式：白色背景，阴影，图标 + 文案"筛选"
- 点击展开浮层：三个选项（全部 / 积分宝箱 / 商家宝箱），单选，选中项高亮

**E6: 定位按钮**
- 位置：右下角，与 E5 同高度，距右边距 16px
- 尺寸：44×44 圆形按钮
- 样式：白色背景，阴影，定位图标居中
- 点击：地图回到以用户当前位置为中心

**E7: 开箱动画（叠加层）**
- 触发：点击可开启状态的宝箱
- 动画：宝箱原位播放开盖动画 0.8s（盖子向后翻开 + 金色光效从箱内溢出）
- 稀有宝箱：额外粒子特效（金色/紫色光点四散，持续 0.5s）
- 动画期间地图不可交互

**E8: 积分奖励卡片（Bottom Sheet）**
- 触发：积分宝箱开箱动画结束后
- 位置：从底部滑出，高度约屏幕 1/4
- 样式：白色背景，顶部圆角 16px，顶部居中拖拽条
- 内容布局：
  - 居中金币图标 48×48
  - 大字"+{n} 积分"，字号 32px，加粗，色值主题金 #FFB800
  - 小字"当前共 {total} 积分"，字号 14px，色值 #999
- 关闭：点击卡片外区域 / 下滑拖拽
- 弹出动画 0.3s ease-out

**E9: 商家券奖励卡片（Bottom Sheet）**
- 触发：商家宝箱开箱动画结束后
- 位置：从底部滑出，高度约屏幕 1/3
- 样式：同 E8 白色背景圆角
- 内容布局（从上到下）：
  - 商家 logo（圆形 40×40）+ 商家名称（字号 16px，加粗）
  - 券面信息大字（如"满50减10"），字号 24px，加粗，色值 #FF3B30
  - 有效期，字号 13px，色值 #999
  - 额外积分奖励标签"+{n} 积分"，胶囊样式，背景 #FFF8E1
  - 底部两个按钮：
    - "查看详情"— 主按钮，主题色背景，白色文字，宽度 60%
    - "关闭"— 文字按钮，色值 #999
- 关闭方式同 E8

**E10: 空状态（叠加层）**
- 触发：1km 范围内无可领取宝箱 / 城市未开放
- 位置：地图中央
- 内容：插画（120×120）+ 文案（字号 15px，色值 #999），居中对齐
- 文案：附近无宝箱 →"附近暂无宝箱，换个地方遛遛看"；城市未开放 →"当前区域暂未开放寻宝，敬请期待"

**E11: 定位权限引导页（全屏覆盖）**
- 触发：定位权限未授权
- 全屏白色背景
- 内容居中：插画（160×160）+ 标题"寻宝需要定位权限"（字号 18px，加粗）+ 说明文案（字号 14px，色值 #666）+ "去设置"按钮（主按钮样式）

### 动画规格

| 动画 | 时长 | 缓动 | 描述 |
|------|------|------|------|
| 宝箱入场（每日首次） | 1.5s 总时长 | ease-out | 宝箱依次从地图"蹦出"，每个间隔 50ms |
| 宝箱弹跳（可开启） | 0.6s 循环 | ease-in-out | 上下位移 4px |
| 稀有光环脉冲 | 1.5s 循环 | ease-in-out | 光环 opacity 在 60%-100% 间脉冲 |
| 开箱动画 | 0.8s | ease-out | 盖子翻开 + 光效溢出 |
| 宝箱消失（被抢） | 0.5s | ease-out | 渐隐 + 粒子四散 |
| 奖励卡片弹出 | 0.3s | ease-out | 底部滑入 |
| 提示条出现 | 0.2s | ease-out | 顶部滑入 |

### 交互规则

| 操作 | 响应 |
|------|------|
| 点击不可开启宝箱 | 无响应（不弹提示，避免打扰） |
| 点击变灰宝箱 | 底部 toast "今日积分宝箱已领完，明天再来" |
| 点击已领取商家宝箱 | 无响应 |
| 地图拖动 / 缩放 | 标准地图手势，实时加载可见范围内宝箱 |
| 开箱动画播放中 | 地图锁定不可交互，防误触 |
| 手机震动 | 进入 50m 范围时短促震动一次（UIImpactFeedbackGenerator.medium） |

**E2: 积分总览卡片**

- 列表项布局：

    - 积分宝箱：木质宝箱缩略图
    - 商家宝箱：金色宝箱缩略图

    - 连续签到：日历图标
  - 中间文案区：
    - 主文案：来源名称，字号 15px，色值 #333
    - 商家宝箱额外显示商家名称（如"商家宝箱·萌宠屋"）
    - 副文案：时间，字号 13px，色值 #999
    - 时间格式：今天/昨天 + HH:mm；更早显示 MM-DD HH:mm
  - 右侧积分变动："+{n}"，字号 17px，加粗，色值主题金 #FFB800
- 列表项之间分割线：左侧缩进 60px，色值 #F5F5F5
- 按日期倒序排列
- 上拉加载更多，每页 20 条

**E5: 底部占位文案**
- 位置：列表滚动到底部后显示
- 文案："积分商城即将上线，敬请期待"
- 字号 13px，色值 #CCC，居中
- 上下间距 24px

**E6: 空状态**
- 触发条件：无任何积分记录

- 内容：插画（100×100）+ 文案"还没有积分记录，去开宝箱吧"，字号 14px，色值 #999

|------|------|

| 上拉到底 | 加载更多流水记录 |
| 点击返回 | 返回寻宝地图 |
| 左滑手势 | 返回寻宝地图（iOS） |

## 页面 4/4：券详情

### 页面信息
