# 我的券包

## 用途

二级页面，管理已领取的商家优惠券。按"未使用"和"已过期"分 Tab 展示。从寻宝地图顶部状态栏券包图标进入。

## 导航

- **来源：** 寻宝地图（状态栏券包图标）
- **去向：** 券详情（点击券卡片或"去使用"按钮）、返回寻宝地图

## 元素

### 导航栏
- 系统安全区 + 44px 高，白色背景
- 左：返回箭头 ‹，44×44 点击区域
- 中："我的券包" 17px 加粗
- 进入时自动清除券包图标红点
- 返回方式：箭头点击、左滑手势（iOS）/ 系统返回键（Android）

### 状态 Tab 栏
- 紧贴导航栏，44px 高，白色背景
- 两个 Tab："未使用({n})" / "已过期({n})"
- 选中态：文字 #FF3B30，底部指示条 3px 高 40px 宽
- 未选中：#999
- 支持左右滑动切换

### 券卡片 — 未使用
- 16px 水平边距，12px 间距
- 白色背景，圆角 12px，阴影 sm
- 左侧券面（宽 90px）：渐变红背景（#FF5E54→#FF3B30），上下半圆锯齿裁切，白色大字居中（"满50\n减10"等）
- 右侧信息：
  - 商家 logo 28×28 圆形 + 名称 15px 加粗
  - 使用条件 13px #666
  - 有效期 12px #999（3 天内变 #FF3B30"即将过期"）
  - "去使用"按钮：红色渐变背景，白字，圆角 16px
- 按下反馈：整卡缩放 0.98

### 券卡片 — 已过期
- 同未使用布局
- 左侧背景变 #CCC 灰
- 右侧文字全部变 #CCC
- 无"去使用"按钮，改为"已过期"文字标签
- 不可点击

### 空状态
- 未使用 Tab：🎫 + "还没有优惠券，去开商家宝箱吧"
- 已过期 Tab：🎫 + "没有已过期的优惠券"

## 状态

- **默认：** 显示"未使用"Tab 下的券列表
- **加载中：** 券卡片骨架屏
- **空状态：** 对应 Tab 无券时显示
- **错误 — 网络：** Toast 提示

## 设计风格

PawQuest 自定义品牌设计系统。券卡片采用票券美学（半圆锯齿裁切），红色券面与白色信息区形成鲜明对比。已过期券通过灰度处理明确区分状态。整体维持轻盈圆润的卡片列表风格，暖白底色（#F7F6F3）。iOS 与 Android 视觉完全统一，仅导航交互跟随平台惯例。字体：PingFang SC（iOS）/ Noto Sans SC（Android）。
