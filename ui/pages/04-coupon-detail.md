# 券详情

## 用途

三级页面，展示优惠券完整信息（券面、使用规则、商家信息），引导用户到店核销。从券包列表或开箱商家券奖励卡片进入。

## 导航

- **来源：** 我的券包（点击券卡片）、寻宝地图（商家券奖励卡片"查看详情"）
- **去向：** 返回来源页面、系统地图（导航）、系统拨号（电话）

## 元素

### 导航栏
- 系统安全区 + 44px 高，白色背景，底部 0.5px 分割线
- 左：返回箭头 ‹，44×44 点击区域
- 中："券详情" 17px 加粗
- 返回方式：箭头点击、左滑手势（iOS）/ 系统返回键（Android）

### 券面卡片（票券造型）
- 16px 边距，白色背景，圆角 16px，阴影 md
- 左右两侧 60% 高度处各一个 16px 半圆裁切（模拟票券撕裂）
- 裁切位置横向虚线（2px dashed #F0F0F0）
- 居中内容：
  - 商家 logo 48×48 圆形 + 名称 18px 加粗
  - 券面核心 36px 加粗 #FF3B30（DIN Alternate 数字），如"满50减10"
  - 有效期 14px #999（3 天内变 #FF3B30 + "即将过期"前缀）

### 使用规则
- 标题"使用规则" 16px 加粗
- 白色容器，圆角 12px，阴影 sm，内边距 16px
- 圆点前缀列表，14px #666，行高 1.8

### 商家信息
- 标题"商家信息" 16px 加粗
- 白色容器，圆角 12px，阴影 sm，内边距 16px
- 地址行：📍 + 地址文案 14px #333
- 电话行：📞 + 号码 14px #007AFF（可点击拨号）
- 导航按钮：右对齐，#FF3B30 描边，"🧭 导航"
- 导航跳转：iOS 支持 AMap / Baidu / Apple Maps，Android 支持 AMap / Baidu / Google Maps

### 底部操作栏（固定）
- 固定底部，白色背景，顶部分割线
- "到店出示此券"按钮：全宽，48px 高，红色渐变，白字 17px 加粗，圆角 24px，阴影 colored

### 券码展示弹层（全屏）
- 点击底部按钮触发
- 白色全屏，屏幕亮度调最大（平台屏幕亮度 API）
- 商家名 16px + 券面 24px 红色（DIN Alternate 数字）+ 二维码 200×200 / 数字码 32px 等宽
- "请出示给商家扫码核销"提示
- 右上角关闭按钮 ✕

## 状态

- **默认：** 展示券完整信息和"到店出示此券"按钮
- **加载中：** 券面卡片骨架屏
- **已过期：** 券面卡片灰色遮罩 + "已过期"水印（20px #999 旋转 -15°），底部按钮变灰不可点击 → "优惠券已过期"
- **错误 — 网络：** Toast 提示

## 设计风格

PawQuest 自定义品牌设计系统。票券美学的高潮页面——半圆裁切 + 虚线分割营造实体优惠券感。信息分区明确（券面 → 规则 → 商家）。视觉基调轻盈圆润，暖白底色（#F7F6F3），柔和阴影。红色 CTA 按钮固定底部保证核销转化。券码弹层极简白底突出二维码。iOS 与 Android 视觉完全统一，仅导航交互跟随平台惯例。字体：PingFang SC（iOS）/ Noto Sans SC（Android），数字用 DIN Alternate。
